Smart_calc

О программе:

Smart_calc позволяет производить стандартные математические расчеты и строить 2-мерные графики.

Установка

Для того, чтобы начать использовать Smart_calc, нужно использовать установить,
выполнив make install в каталоге src

Удаление

Для удаления программы вы можете использовать команду make uninstall

Как начать работу с программой?

    • Перейти в каталог smart_calc/src;
    • Ввести в консоль команду make;
    • После установки программы откроется окно калькулятора;
    • Для построения графика необходимо ввести функцию, задать min/max значения по оси X, и нажать кнопку "graph";

Программа позволяет:

    • Производить стандартные математические операции:
      • - Сложение: +;
      • - Вычитание: -;
      • - Умножение: *;
      • - Деление: /;
      • - Возведение в степень: ^;
      • - Нахождение квадратного корня: sqrt(x);
      • - Нахождение остатка от деления на число: a mod b;
      • - Вычисление натурального логарифма: ln(x);
      • - Вычисление десятичного логарифма: log(x);
      • - Реализована приоритетность математических операций.
    • Построение графиков функций с возможностью указать максимальные и минимальные значение аргумента Х:
      • sin(x);
      • cos(x);
      • tan(x)
      • asin(x);
      • acos(x);
      • atan(x)

Технические требования для запуска программы:

    • Установленный и настроенный фреймворк QtCreator версии 5.12;
    • Операционная система Linux (возможен запуск на Mac Os).
